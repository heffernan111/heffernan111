🌌 HabOS - Analogue Mission Control

A mission control interface for analogue astronaut simulations, built with Go and Wails. Supports live telemetry via Redis and WebSockets, weather monitoring, and a time-delay communication system to simulate interplanetary conditions. Includes fallback safety protocols and modular integration for external research feeds.

Go Wails WebSocket Redis Mission Control

Cloud-based control panel for robotic telescopes, built for the Astrophysics Research Institute. Includes hardware state views, environment sensors, queue-based automation, and live WebSocket telemetry. Deployed to observatory systems and tested in La Palma.
